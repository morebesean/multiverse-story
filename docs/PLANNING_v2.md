# Multiverse - 새로운 서비스 기획 아이디어 (v2)

"멀티버스" 키워드만 남기고 처음부터 기획한 5가지 방향.

---

## 기획 A: "Multiverse Chat" — 평행우주의 나와 대화하기

**컨셉:** 다른 선택을 한 "나"를 AI로 생성하고, 그 페르소나와 실시간 채팅

**플로우:**
1. 간단한 프로필 입력 (이름, 나이대, 분기점 선택)
2. AI가 "평행우주의 나" 페르소나를 생성 (로딩 중 광고)
3. 채팅 인터페이스에서 대화 시작
4. 대화 내용 중 인상적인 문장을 카드로 만들어 공유

**강점:**
- 1회성이 아닌 지속적 인터랙션 → 체류 시간 극대화
- 대화 중간중간 자연스러운 광고 삽입 가능
- 감정적 연결이 강해서 공유 욕구 높음
- "다른 분기점의 나"를 여러 명 만들 수 있어 재방문 유도

---

## 기획 B: "Multiverse Swap" — 두 사람의 인생을 교환

**컨셉:** 친구 2명이 각자 프로필을 입력하면, 서로의 인생을 교환한 평행우주 이야기를 생성

**플로우:**
1. 내 프로필 입력 → 고유 링크 생성
2. 친구에게 링크 공유
3. 친구가 자기 프로필 입력
4. AI가 "내가 친구의 인생을 살았다면 / 친구가 내 인생을 살았다면" 2편 생성

**강점:**
- 바이럴 내장 구조 — 반드시 2명이 참여해야 완성
- 친구와 비교하는 재미
- 자연스러운 공유 동기

---

## 기획 C: "Multiverse Timeline" — 선택지 시뮬레이터

**컨셉:** 인생의 주요 갈림길에서 다른 선택을 할 때마다 타임라인이 분기되는 인터랙티브 스토리

**플로우:**
1. 시작점 설정 (나이, 상황)
2. AI가 첫 번째 갈림길 제시 (선택지 A vs B)
3. 선택할 때마다 타임라인이 분기 → 트리 구조로 시각화
4. 3~5번의 선택 후 최종 결과 도달
5. 모든 분기점을 한눈에 볼 수 있는 타임라인 트리 제공

**강점:**
- 게임적 요소로 재미와 몰입감 극대화
- 각 선택지마다 광고 삽입 가능 (3~5회)
- "다른 선택으로 다시 해보기" → 자연스러운 재플레이
- 트리 구조 이미지가 SNS에서 매력적

---

## 기획 D: "Multiverse Match" — 나와 가장 다른 평행우주의 나 찾기

**컨셉:** 사용자 프로필 기반으로 "가장 극단적으로 다른 인생"을 살고 있는 멀티버스의 나를 매칭

**플로우:**
1. 현재 나의 프로필 입력
2. AI가 "정반대의 나" 생성 (로딩 중 광고)
3. 비교 카드: 현실의 나 vs 멀티버스의 나
4. "정반대의 나"가 보내는 편지/메시지
5. 매칭 호환도 점수

**강점:**
- MBTI 테스트처럼 자기 발견 욕구 자극
- 비교 카드가 시각적으로 공유에 최적화

---

## 기획 E: "Multiverse Diary" — 평행우주의 나의 일기

**컨셉:** 매일 하나씩, 평행우주의 내가 쓴 일기를 받아보는 서비스

**플로우:**
1. 최초 1회 프로필 + 분기점 설정
2. 매일 알림으로 "오늘의 일기" 도착
3. 일기에 대해 코멘트/반응 가능
4. 일주일치 모으면 "주간 리포트" 생성

**강점:**
- DAU 확보에 최적
- 매일 새로운 콘텐츠 → 매일 광고 노출
- 구독 모델 전환 가능성
